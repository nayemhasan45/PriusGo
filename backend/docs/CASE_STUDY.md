# PriusGo Case Study

## Problem

Al-Amin needed a simple way to manage Toyota Prius rentals in Šiauliai without juggling messages, paper notes, and manual availability checks.

## Solution

PriusGo is a Next.js + Supabase rental app with:

- public fleet pages with live availability
- authenticated booking requests
- admin booking and car management
- safe overlap prevention at the database level
- mobile-friendly layout for customer and admin use

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Supabase Auth, Postgres, RLS, Storage
- Vitest
- Vercel

## Screenshots

- Homepage and fleet view
- Booking form
- Admin bookings dashboard
- Admin cars dashboard

## Business Impact

- Faster booking review
- Less double-booking risk
- Clearer rental rules for customers
- Better day-to-day fleet tracking
- Ready for production deployment and future V2 growth
